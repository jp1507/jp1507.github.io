## Chapter 11. 상속 : 상속받은 함수 감추기

### 접근 지정자 바꾸기 : using

---

```cpp
#include <iostream>
using namespace std;

class Base
{
protected:
    int m_i;
public:
    Base(int value) : m_i(value) {}

    void print()
    {
        cout << "I'm Base " << endl;
    }
};

class Derived : public Base
{
private:
    double m_d;
public:
    Derived(int value) : Base(value) {}

    using Base::m_i;
};

int main()
{
    Derived derived(7);
    derived.m_i = 1024;
    return 0;
}
```

- 자식 클래스의 `public` 영역 내에 `using Base::m_i`와 같이 명시해주면 `m_i`가 `public`으로 접근 지정자가 변경됨

```cpp
class Derived : public Base
{
public:
    Derived(int value) : Base(value) {}
private:
    using Base::print;
};

int main()
{
    Derived derived(7);
    derived.print();    // Error !!!
    return 0;
}
```

- `private` 영역에 `using Base::print`로 함수 이름만 명시해주어 `print()` 함수가 `private` 접근 지정자로 변경되었기 때문에 호출 시 에러가 발생함

### 상속 받은 함수 접근 방지 : delete

---

```cpp
#include <iostream>
using namespace std;

class Base
{
protected:
    int m_i;
public:
    Base(int value) : m_i(value) { }

    void print()
    {
        cout << "I'm Base " << endl;
    }
};

class Derived : public Base
{
private:
    double m_d;
public:
    Derived(int value) : Base(value) {}
private:
    void print() = delete;
};

int main()
{
    Derived derived(7);
    derived.print();    // Error !!!
    derived.Base::print();
    return 0;
}
```

- 자식 클래스에서 부모 클래스로부터 상속받은 멤버 함수 `print()`를 `=delete`로 호출할 수 없게 막을 수 있음
  - 따라서 `base.print()`는 가능하지만 `derived.print()`는 호출 불가능
