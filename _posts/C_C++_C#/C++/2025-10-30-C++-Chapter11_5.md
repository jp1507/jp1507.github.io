## Chapter 11. 상속 : 자식 클래스에 새로운 기능 추가하기

- 자식 클래스에서 부모 클래스로부터 상속받은 멤버에 접근하여 사용하려고 할 때

### 1. 상속되는 멤버를 private로 설정

---

```cpp
class Base
{
private:
    int m_value;
public:
    Base(int value) : m_value(value) {}

    int & getValue() 
    {
        return m_value;
    }
};

class Derived : public Base
{
public:
    Derived(int value = 0) : Base(value) { }

    void setValue(int value)
    {
        Base::getValue() = value;
    }
};

int main()
{
    Derived d(5);
    d.setValue(2);
    cout << d.getValue() << endl;    // 2 출력

    return 0;
}
```

- `m_value`를 `private`으로 설정한 경우 자식 클래스에서 이를 접근해 조작하기 위해서 setter, getter 접근 함수를 반드시 구현해야 함

### 2. 상속되는 멤버를 protected로 설정

---

```cpp
class Base
{
  protected:
      int m_value;

  public:
      Base(int value)
          : m_value(value)
      {}
};

class Derived : public Base
{
public:
    Derived(int value = 0)
         : Base(value)
    {
    }

    void setValue(int value)
    {
      Base::m_value = value;
    }
};

int main()
{
    Derived d(5);
    
    d.setValue(2);
    
    // cout << d.m_value() << endl;

    return 0;
}
```

- `m_value`를 `protected`로 설정한 경우 자식 클래스 내에 `m_value`에 바로 접근 가능
  - 자식 클래스에서 바로 접근 가능하지만 외부에서 자식 객체로는 접근할 수 없음

함수를 거치지 않고 바로 접근할 수 있기 때문에 최적화를 위해서는 `protected`로 설정하는 것을 권장
