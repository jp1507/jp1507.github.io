## Chapter 8. 객체 지향의 기초 : 클래스 안에 포함된 자료형

### 클래스 내부에서의 중첩 자료형 사용

---

```cpp
class Fruit
{
public:
    enum FruitType_1
    {
        APPLE, BANANA, CHERRY,
    };
    enum class FruitType_2
    {
        APPLE, BANANA, CHERRY,
    };
    class InnerClass { };
    struct InnerStruct { };

private:
    FruitType_1 type_1;
    FruitType_2 type_2;
    InnerClass c;
    InnerStruct s;
}
```

### 클래스 밖에서의 중첩 자료형 사용

---

```cpp
int main()
{
    Fruit::BANANA;  // enum 
    Fruit::FruitType_2::BANANA;  // enum class 
    Fruit::InnerClass c;  // inner-class
    Fruit::InnerStruct s; // inner-struct
}
```
