## 포인터와 참조의 멤버 선택

---

- 포인터 혹은 참조로 구조체와 클래스의 멤버에 접근할 수 있음

```cpp
#include<iostream>
using namespace std;

struct Person
{
  int age;
  double weight;
};

int main()
{
  Person person;
  person.age = 5;

  Person &ref = person;
  ref.age = 15;

  Person *ptr = &person;
  ptr->age = 30; // 포인터로 멤버 접근 시 화살표(->) 사용
  (*ptr).age = 20; // .가 *보다 우선순위가 높기 때문에 간접참조를 하기 위해 괄호 사용

  Person &ref2 = *ptr;
  ref2.age = 45;

  cout << &person << endl;
  cout << &ref2 << endl;

  return 0;
}
```
