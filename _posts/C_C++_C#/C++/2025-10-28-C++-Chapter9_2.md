## Chapter 9. 연산자 오버로딩 : 산술 연산자 오버로딩

- 산술 연산자 : `+`, `-`, `/`, `*`, `%`

### 전역 함수로 구현

---

```cpp
class Cents
{
private:
    int m_cents;
public:
    Cents(int cents = 0) { m_cents = cents; }

    friend void operator +(const Cents& c1, const Cents& c2);  
};

void operator + (const Cents& c1, const Cents& c2)
{
    cout << c1.m_cents + c2.m_cents << endl;
}

int main()
{
    Cents cents1(5);
    Cents cents2(7);

    cents1 + cents2;    // 12 출력
}
```

### 멤버 함수로 구현

---

```cpp
class Cents
{
private:
    int m_cents;
public:
    Cents(int cents = 0) { m_cents = cents; }
    void operator +(const Cents& c) 
    {
        cout << m_cents + c.m_cents << endl;
    }
};

int main()
{
    Cents cents1(5);
    Cents cents2(7);

    cents1 + cents2;    // 12 출력
}
```
