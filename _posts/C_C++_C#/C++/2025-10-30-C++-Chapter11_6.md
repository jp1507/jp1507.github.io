## Chapter 11. 상속 : 상속받은 함수 오버라이딩하기

### 상속받은 함수 오버라이딩하기

---

**단순 상속의 경우**  

```cpp
#include <iostream>
using namespace std;

class Base
{
protected:
    int m_i;
public:
    Base(int value) : m_i(value) { }

    void print()
    {
        cout << "I'm Base " << endl;
    }
};

class Derived : public Base
{
private:
    double m_d;
public:
    Derived(int value) : Base(value) {}
};

int main()
{
    Base base(5);
    Derived derived(7);  

    base.print();   // "I'm Base" 출력
    derived.print();    // "I'm Base" 출력
}
```

**상속받은 함수를 오버라이딩한 경우**  

```cpp
#include <iostream>
using namespace std;

class Base
{
protected:
    int m_i;
public:
    Base(int value) : m_i(value) { }

    void print()
    {
        cout << "I'm Base " << endl;
    }
};

class Derived : public Base
{
private:
    double m_d;
public:
    Derived(int value) : Base(value) {}

    void print()
    {
        cout << "I'm Derived" << endl;
    }
};

int main()
{
    Base base(5);
    Derived derived(7);  

    base.print();   // "I'm Base" 출력
    derived.print();    // "I'm Derived" 출력
}
```

- C++에서 멤버를 호출할 때, 변수의 타입이 무엇이냐에 따라 맞는 타입의 멤버를 호출

**오버라이딩하더라도 부모 멤버 함수를 호출하고 싶은 경우**  

```cpp
friend std::ostream & operator <<(std::ostream& out, const Derived& d)
{
    cout << static_cast<Base>(d) << endl;   // "I'm Base" 출력
    cout << "I'm Derived" << endl;  // "I'm Derived" 출력
    return out;
}
```
