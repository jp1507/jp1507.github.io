## std::vector

---

```cpp
std::vector <int> array = {1, 2, 3, 4, };
```

- `#include <vector>`
- 동적 배열과 비슷함
- 크기를 런타임에 결정할 수 있음
  - 초기화 시 모든 원소를 넣을 필요가 없음
- `vector`의 크기를 `resize()`로 아무 시점에 바꿀 수 있음
- 메모리를 알아서 해제해줌
- `vector`의 길이에 대한 정보를 담고 있음
