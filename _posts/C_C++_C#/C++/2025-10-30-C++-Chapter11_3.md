## Chapter 11. 상속 : 자식 클래스들의 생성과 초기화

### 크기 비교

---

- 부모 클래스를 자식 클래스가 상속받을 때 두 타입의 객체 크기 비교
  - Padding을 고려해야 함

```cpp
class Mother
{
    int m_i;
};
class Child
{
    int m_d;
};

int main()
{
    cout << sizeof(Mother) << endl;   // 4 출력 (int)
    cout << sizeof(Child) << endl;    // 8 출력 (int + int)
}
```

```cpp
class Mother
{
    int m_i;
};
class Child
{
    double m_d;
};

int main()
{
    cout << sizeof(Mother) << endl;   // 4 출력 (int)
    cout << sizeof(Child) << endl;    // 16 출력 (int + double)
}
```

### 생성자 호출은 부모부터, 소멸자 호출은 자식부터

---

```cpp
#include <iostream>
using namespace std;

class A
{
public:
    A(int a)
    {
        cout << "Constructor A"<< endl;
    }
    ~A()
    {
        cout << "Destructor A " << endl;
    }
};
class B : public A
{
public:
    B(int a, double b) : A(a)
    {
        cout << "Constructor B" << endl;
    }
    ~B()
    {
        cout << "Destructor B " << endl;
    }
};
class C : public B
{
public:
    C(int a, double b, char c) : B(a, b)
    {
        cout << "Constructor C" << endl;
    }
    ~C()
    {
        cout << "Destructor C " << endl;
    }
};

int main()
{
    C c(1024, 3.14, 'a');
    return 0;
}
```
```text
- 출력 -
Constructor A
Constructor B
Constructor C
Destructor C
Destructor B
Destructor A
```
