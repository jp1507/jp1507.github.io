## Chapter 14. 예외처리 : 예외처리의 기본

- 잘못된 결과가 도출되면 `return -1`, 혹은 `bool` 타입의 플래그를 두어 잘못되면 `if(!flag)` 방식으로 예외 처리를 직접 하는 방법이 있음
- Real time과 같은 성능이 중요한 부분의 개발에서는 위와 같이 직접 처리하고 서버 다운과 같은 예측할 수 없는 부분이거나 성능 면에 있어서 비교적 속도가 느리더라도 괜찮은 부분은 간단하고 안정적인 예외 처리 기능을 사용하는 것이 좋음

### try-throw-catch

---

```cpp
#include <iostream>

using namespace std;

int main()
{
    double x;
    cin >> x;
    
    try
    {
        if (x < 0.0) 
            throw string("Negative input");
        
        cout << std::sqrt(x) << endl;
    }
    catch (string error_message)
    {
        cout << error_message << endl;
    }

    return 0;
}
```

- `try`
  - 예외가 발생하지 않으면 `try` 부분만 실행됨
- `throw`
  - `catch`에게 에러 메세지를 던지며, 예러가 발생한 부분
- `catch`
  - `throw`가 던진 에러 메세지를 받아 예외 발생에 대한 대응을 처리함

### 주의사항

---

- 예외 처리는 자동 형변환을 지원하지 않으며 데이터 타입에 대해 매우 엄격함

```cpp
int main()
{
    double x;
    cin >> x;
    
    try
    {
        if (x < 0.0) 
            throw "Negative input";
        
        cout << std::sqrt(x) << endl;
    }
    catch (string error_message)
    {
        cout << error_message << endl;
    }

    return 0;
}
```

- `string("Negative input")`은 `string` 타입이고, `"Negative input"`은 `const char*` 타입
- 자동 형변환이 되지않아 `string` 타입인 `error_message`로 `const char*` 타입인 `"Negative input"`을 받으면 에러가 발생

**throw가 던진 것에 대해 타입이 알맞는 catch가 실행**  

```cpp
int main()
{
    try
    {
        throw -1;
    }
    catch (int x)
    {
        cout << "Catch Integer " << x << endl;
    }
    catch (string error_message)
    {
        cout << error_message << endl;
    }

    return 0;
}
```
```text
- 출력 -

Catch Integer -1
```

- `throw`에서 `-1`을 던지므로 `catch (int x)`가 실행됨

```cpp
int main()
{
    try
    {
        throw "My error message";
    }
    catch (double x)
    {
        cout << "Catch double" << endl;
    }
    catch (const char* error_message)
    {
        cout <<  "const char* " << error_message << endl;
    }

    return 0;
}
```
```text
- 출력 -

const char* My error message
```

- `throw`에서 `"My error message"`를 던지므로 `catch (const char* error_message)`가 실행됨
