## Chapter 7. 함수

### 매개변수(parameter)와 인자(argument) 구분

---

- **함수**는 인자(argument)가 매개변수(parameter)에 전달되는 형태로 동작함

```cpp
void func(int x, int y) // int x, int y는 매개변수(parameter)
{
  // ...
}
int main()
{
    func(1, 2);  // 1, 2는 인자(argument)
}
```
